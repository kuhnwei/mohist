# simple-snake
JavaScript 实现一个简单的贪吃蛇小游戏。

## 操作指南

任意方向键开始游戏

键盘 UP ARROW 键(↑)：向上移动

键盘 DOWN ARROW 键(↓)：向下移动

键盘 LEFT ARROW 键(←)：向左移动

键盘 RIGHT ARROW 键(→)：向右移动

键盘Esc：结束本局游戏
