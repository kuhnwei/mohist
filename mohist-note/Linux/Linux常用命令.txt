文件处理命令：
	命令 [-选项] [参数]
	例：ls -la /etc
	
	目录处理命令：ls
	命令名称：ls	愿意：list
		-a 显示所有文件，包括隐藏文件
		-l 详细信息显示
		-d 查看目录属性
	-	rw-	r--	r--
	开头"-":表示文件
	开头"d":表示目录
	权限：	u所有者	g所属组	o其他人
		r读	w写	x执行	


	tail -f filename 查看文件（常用于查看正在活跃的日志文件）

目录处理命令：
	mkdir
	命令英文原意：make directories
	语法：mkdir -p [目录名]
	功能：创建新目录
		-p 递归创建
	范例：mkdir -p /tmp/Japan/boduo
	
	cd
	命令英文原意：change directory
	语法：cd [目录]
	功能：切换目录
	
	pwd
	命令英文原意：print working directory
	语法：pwd
	功能：显示当前目录
	
	rmdir
	命令英文原意：remove empty directories
	语法：rmdir [目录名]
	功能：删除空目录
	
	cp
	命令英文原意：copy
	语法：cp -rp [原文件或目录][目标目录]
		-r 复制目录
		-p 保留文件属性
	功能：负责文件或目录

	mv
	命令英文原意：move
	语法：mv [原文件或目录] [目标目录]
	功能描述：剪切文件、改名
	
	rm
	命令英文原意：remove
	语法：ra -rf [文件或目录]
		-r 删除目录
		-f 强制执行
	功能：删除文件

	touch
	功能：创建空文件
