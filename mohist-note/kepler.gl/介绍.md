# 介绍

Kepler.gl专为地理空间数据分析而设计。它允许技术和非技术受众可视化城市或地区趋势。使用Kepler.gl，你可以在浏览器中可视化大量位置数据。随着时间的推移播放地理时间趋势。探索，过滤并深入了解位置数据，以获得洞察力。

![1](./img/1.png)

本指南将教你如何通过向地图添加数据，创建图层，添加过滤器等来在Kepler.gl中执行数据分析。
